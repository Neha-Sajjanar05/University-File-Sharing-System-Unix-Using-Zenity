# University-File-Sharing-System-Unix-Using-Zenity
A role-based file sharing system built with Bash and Zenity GUI. Features secure admin/student authentication, file management operations, download tracking, and activity logging. Perfect for university shell programming projects demonstrating user authentication, file I/O, and GUI integration in Unix environments.
Features:
Dual User Roles: Separate admin and student accounts with different permissions User Management: Registration and login system for both admin and student users Student Operations: Download files and read content within the application Admin Operations: Full file management (add, delete, read) plus download tracking Activity Logging: Track and monitor student file downloads GUI Interface: User-friendly dialogs using Zenity for seamless interaction
Requirements:
Linux/WSL environment Zenity package installed Bash shell
Usage:
chmod +x file_sharing.sh ./file_sharing.sh
